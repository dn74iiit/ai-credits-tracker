Sync Instructions

This dashboard pulls data directly from a local repository configuration. To set up automatic sync from your laptop:

  1. Configure the API keys inside your local .env file.
  2. Verify by running python sync_credits.py in your project folder.
  3. Set up a scheduler to run the script automatically in the background.

Configuration Guide

Local environment variables (.env)

Ensure your .env file is in the root directory and contains the keys:

OPENAI_API_KEY=sk-...
GEMINI_API_KEY=AIzaSy...
DEVIN_API_KEY=cog_...

Automated Scheduled Task on Windows

To refresh your stats automatically every 10 minutes:

  1. Open Task Scheduler.
  2. Click Create Basic Task. Name it AI Credits Sync.
  3. Set Trigger to Daily, then edit the task properties to: "Repeat task every 10 minutes".
  4. Set Action to Start a Program:
    • Program/script: pythonw.exe (runs without popping up a console window)
    • Add arguments: sync_credits.py
    • Start in: c:\Users\dhanu\OneDrive\Documentos\AI plans tracker site
